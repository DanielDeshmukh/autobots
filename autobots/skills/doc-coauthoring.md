# Document Co-Authoring Skill

## Purpose
Expert in collaborative document creation, editing, and review. Facilitates multi-author workflows, version control, and document quality assurance.

## Core Capabilities

### 1. Document Structure
- Template design
- Section organization
- Cross-referencing
- Index generation

### 2. Content Management
- Version tracking
- Change tracking
- Comment management
- Review workflows

### 3. Quality Assurance
- Style consistency
- Grammar checking
- Citation management
- Plagiarism detection

## Collaboration Patterns

### Review Workflow
```
1. Author creates draft
2. Reviewer A adds comments
3. Reviewer B suggests changes
4. Author addresses feedback
5. Final approval
6. Publication
```

### Version Control
```markdown
## Version History
- v1.0 (2024-01-15): Initial draft
- v1.1 (2024-01-20): Incorporated feedback
- v1.2 (2024-01-25): Final revisions
```

## Common Workflows

### Document Creation
```
1. Define requirements
2. Create outline
3. Write draft
4. Self-review
5. Peer review
6. Revisions
7. Final approval
8. Publication
```

### Major Revision
```
1. Collect feedback
2. Prioritize changes
3. Implement revisions
4. Verify changes
5. Update version
6. Distribute updated version
```
