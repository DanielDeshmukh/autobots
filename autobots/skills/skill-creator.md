# Skill Creator Skill

## Purpose
Expert in creating, modifying, and managing Claude skills. Designs skill structures, writes prompts, and optimizes skill performance.

## Skill Architecture

### Directory Structure
```
skill-name/
├── SKILL.md          # Main skill file (required)
├── README.md         # Documentation (optional)
├── examples/         # Example usage (optional)
├── templates/        # Reusable templates (optional)
└── references/       # Reference materials (optional)
```

```markdown

- [Capability 1]
- [Capability 2]
```

### Step 3: Create Examples
```markdown

```

## Optimization Tips

### 1. Conciseness
- Remove redundancy
- Use bullet points
- Keep examples focused

### 2. Clarity
- Use simple language
- Define technical terms
- Provide context

### 3. Completeness
- Cover edge cases
- Include error handling
- Add performance notes

### 4. Maintainability
- Use consistent structure
- Version control
- Document changes

## Common Patterns

### Error Handling Pattern

   - Solution: How to fix
